# DanielWJChen.com
This repository stores the project that makes up DanielWJChen.com.

## Installation
To create a environment from scratch, install all the following dependencies.
### NVM and NPM
```
nvm install 8
```
### NPM global packages
```
npm install -g webpack webpack-cli webpack-dev-server
```

### Install NPM
```
npm install
```
